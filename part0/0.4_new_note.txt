Browser->Server: HTTP POST input to /new_note
note over Server: Server recieves post request and adds the note to notes array
note over Server: Server sends a redirect request to browser to /notes address

Server-->Browser: status code 302
note over Browser: Browser sends a a new GET request

Browser->Server: HTTP GET /notes
Server-->Browser: html notes

Browser->Server: HTTP GET main.css
Server-->Browser: main.css

Browser->Server: HTTP GET main.js
Server-->Browser: main.js

note over Browser: Browser starts executing javascript, sends request for data.json

Browser->Server: HTTP GET data.json
Server-->Browser: data.json

note over Browser: Browser executes the event handler that renders notes to display
